# Comandos

Para correr los todos los test use el siguiente comando:
```bash
mvn clean test
```

Y para ejecutar todas las pruebas con un determinado tag use el siguiente comando
```bash
 mvn test -Dkarate.options="--tags @<Tag>"
```
